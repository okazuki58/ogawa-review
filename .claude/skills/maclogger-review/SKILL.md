# maclogger自動振り返りスキル

## 概要

macloggerの日次・週次ログを読み込み、指定期間の振り返りを自動生成します。Value別（Be hungry/Be an owner/Be sustainable）に実績を分類し、`/Users/ogawakazuki/dev/xincere-review/reviews/`にmdファイルで出力します。

## 使用タイミング

- 週次振り返りを作成したい時（毎週金曜など）
- 月次振り返りを作成したい時（月末）
- 半期振り返りを作成したい時（2月末、8月末）
- 目標達成状況を確認したい時

## 主要機能

### 1. 期間指定での振り返り生成

ユーザーが指定した期間のログを読み込み、振り返りを生成します。

**対応期間**:
- **週次**: 先週（月〜金）、今週（月〜今日）
- **月次**: 今月、先月、特定月（例：9月）
- **半期**: 今半期、前半期、特定半期（例：2025H2）

### 2. Value別の実績分類

macloggerのログから実績を抽出し、以下の3つのValueに自動分類：

#### Be hungry（貪欲になれ）
- 技術学習の記録（技術書、AWS、新しい技術）
- 記事執筆・アウトプット
- 新しいパターン・技術の適用
- 勉強会・発表

#### Be an owner（当事者たれ）
- タスク完了・締切遵守
- チーム貢献（コミュニケーション、サポート）
- 問題解決
- 責任完遂

#### Be sustainable（継続的であれ）
- ドキュメント化
- 自動化・効率化
- リファクタリング
- 属人化解消の取り組み

### 3. 振り返りファイルの出力

`/Users/ogawakazuki/dev/xincere-review/reviews/`に以下の形式でファイル出力：

- **週次**: `YYYY-WXX.md`（例：2025-W01.md）
- **月次**: `YYYY-MM.md`（例：2025-09.md）
- **半期**: `YYYY-HX.md`（例：2025-H2.md）

## 使用例

### 週次振り返りの生成

```
ユーザー: 先週の振り返りを作成して
```

→ Claude:
1. macloggerの`/Users/ogawakazuki/dev/maclogger/reports/daily/`から先週のログ（月〜金）を読み込み
2. Value別に実績を分類
3. `/Users/ogawakazuki/dev/xincere-review/reviews/2025-W52.md`に出力

### 月次振り返りの生成

```
ユーザー: 9月の振り返りを作成して
```

→ Claude:
1. macloggerの9月分のログを全て読み込み
2. Value別に実績を分類
3. 目標達成状況を確認（`goals/2025-09-日次目標.md`と照合）
4. `/Users/ogawakazuki/dev/xincere-review/reviews/2025-09.md`に出力

### 半期振り返りの生成

```
ユーザー: 2025H2の振り返りを作成して
```

→ Claude:
1. macloggerの9月〜2月のログを全て読み込み
2. Value別に実績を分類
3. 半期目標との達成状況を比較（`goals/2025-H2-半期目標.md`と照合）
4. `/Users/ogawakazuki/dev/xincere-review/reviews/2025-H2.md`に出力

## 重要な仕様

### macloggerログの場所

- **日次ログ**: `/Users/ogawakazuki/dev/maclogger/reports/daily/YYYY-MM-DD.md`
- **週次ログ**: `/Users/ogawakazuki/dev/maclogger/reports/weekly/YYYY-WXX.md`

### 出力先

- **振り返りファイル**: `/Users/ogawakazuki/dev/xincere-review/reviews/`

### 目標ファイルとの照合

振り返り生成時、以下のファイルと照合して達成状況を確認：

- **週次**: `goals/2025-H2-週次目標.md`
- **月次**: `goals/2025-H2-月次目標.md`
- **半期**: `goals/2025-H2-半期目標.md`

## 実行フロー

1. **期間の確認**: ユーザーが指定した期間を解釈（先週、今月、2025H2など）
2. **ログの読み込み**: macloggerから該当期間のログを全て読み込み
3. **実績の抽出**: ログから具体的な実績を抽出
4. **Value別の分類**: 実績を3つのValueに自動分類
5. **目標との照合**: 目標ファイルと照合し、達成状況を確認
6. **ファイル出力**: 振り返りファイルを生成・保存
7. **ユーザーに報告**: 出力先とサマリーを報告

## Claude Codeへの指示

振り返り生成時は、以下の手順を必ず実行すること：

1. **期間の明確化**: ユーザーが「先週」「今月」などと言った場合、具体的な日付範囲を確認
2. **ログの網羅的な読み込み**: 該当期間のログを全て読み込む（漏れがないように）
3. **事実ベースの記載**: ログに記載されている事実のみを抽出（推測や創作をしない）
4. **Value分類の精度**: 実績が複数のValueに該当する場合、最も適切なValueに分類
5. **目標達成状況の正確な判定**: 目標ファイルと照合し、達成/未達成を正確に判定
6. **ファイル上書き確認**: 既に同じファイル名の振り返りが存在する場合、上書きするか確認

## performance-review skillとの連携

このskillは`performance-review` skillと連携して使用されます：

- **maclogger-review**: ログから振り返りを自動生成
- **performance-review**: 振り返りをもとに評価シートを作成

半期の評価シート作成時は、`reviews/2025-H2.md`を参照して具体的な実績を記載します。
